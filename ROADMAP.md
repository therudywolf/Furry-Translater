## Известные ограничения

- Reasoning-модели на бэкенде «думают» в `reasoning_content`; при их включении
  держите `reasoning_effort=none` и достаточный `LLM_MAX_TOKENS`.
- Прямой режим (`config.js` с `API_KEY`) раскрывает ключ в браузере — только для
  локального теста, не для публичного сервера.
- Прокси резолвит upstream-хост при старте nginx; при смене IP DDNS нужен
  рестарт контейнера (можно перейти на `resolver` + переменную, если важно).
